# Use Cases

## 1. Program brief
Turn a fuzzy initiative into:
- scope
- goals
- milestones
- dependencies
- owners
- risks
- success metrics

## 2. Launch planning
Create:
- workstreams
- timeline
- readiness checklist
- stakeholder comms
- rollback plan

## 3. Weekly status updates
Generate exec-readable updates with:
- status
- changes
- blockers
- decisions needed
- next milestone

## 4. Risk and dependency management
Identify:
- top risks
- cross-team dependencies
- mitigation owners
- escalation paths

## 5. Decision memos
Structure options, tradeoffs, and recommendation clearly.

## 6. Postmortems
Write corrective-action-focused retrospectives that improve systems.

## 7. Portfolio / operating reviews
Summarize progress, slippage, investment choices, and execution risks across initiatives.

## 8. Claude-assisted PM workflows
Use the skill with Claude-compatible agents to improve speed and consistency on recurring program artifacts.
