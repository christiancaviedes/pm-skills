# Scenario Pack

## 1. AI platform launch

Use the suite to produce:
- a program brief for model gateway rollout
- a launch plan across platform, product, support, and GTM
- a RAID review for safety, latency, and vendor dependencies
- a weekly exec update for launch readiness
- a postmortem after rollout

## 2. Marketplace reliability initiative

Use the suite to produce:
- a cross-functional execution plan for reducing failed transactions
- a dependency map across payments, trust, and customer support
- an operating review summary for leadership
- an escalation memo when one partner is on the critical path

## 3. Enterprise migration program

Use the suite to produce:
- a migration brief
- milestone plan
- stakeholder update cadence
- risk review
- go/no-go launch decision memo

## 4. New SaaS onboarding launch

Use the suite to produce:
- onboarding launch plan
- readiness checklist
- KPI definitions
- weekly progress update
- launch comms matrix

## 5. Internal tooling rollout

Use the suite to produce:
- phased rollout strategy
- portfolio review summary
- adoption KPI definitions
- decision memo on scope tradeoffs
