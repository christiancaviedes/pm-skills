# Repo Vision

Build the strongest public repository for PM operating skills.

## Standard

This repo should feel:
- senior
- practical
- crisp
- hard to outgrow
- immediately useful in real work

## What “world-class” means here

- It improves judgment, not just formatting.
- It helps under ambiguity, not only for clean textbook prompts.
- It produces artifacts executives and cross-functional teams can actually use.
- It avoids MBA fog and agile cargo cults.

## Expansion path

Likely future skills:
- product-strategy
- tpm-execution
- exec-communication
- launch-operations
- user-research-synthesis
- metrics-and-instrumentation
- incident-postmortems

## Non-goals

- Massive template sprawl
- Over-specialization to one tool like Jira
- Empty “framework collection” content
- Pretending every PM artifact needs six pages

## Editorial bar

If a section sounds impressive but does not improve decision quality or execution clarity, cut it.
