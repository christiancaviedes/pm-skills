# Benchmarks

This repo aims to outperform generic AI PM outputs on:
- clarity of recommendation
- specificity of owners and milestones
- visibility of risks and dependencies
- quality of executive communication
- usefulness under ambiguity

## Informal benchmark test

Give a model the same fuzzy request twice:
1. without this repo
2. with one of these skills

Compare the outputs on:
- did it lead with the answer?
- did it define the decision?
- did it surface risks?
- did it name owners / milestones?
- did it avoid filler?

That is the standard this suite is designed to win.
