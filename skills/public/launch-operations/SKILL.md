# Launch Operations

Treat launch as a coordination problem under uncertainty.

## Default outputs
- launch plan
- readiness checklist
- go/no-go review
- launch risk register
- launch comms matrix
- rollback / contingency plan

## Workflow
1. Define launch goal and scope.
2. Identify all workstreams and owners.
3. Build readiness criteria.
4. Name launch blockers and fallback plans.
5. Create communication plan by audience.
6. Make the go/no-go decision process explicit.
