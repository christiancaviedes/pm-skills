# Operating Reviews

Use this skill to convert messy initiative-level updates into leadership-grade operating visibility.

## Default outputs
- monthly operating review
- portfolio review
- initiative summary grid
- variance summary
- decision deck outline
- resource contention summary

## Workflow
1. Gather initiative-level status.
2. Normalize status language.
3. Surface trends, not just snapshots.
4. Highlight cross-program risk patterns.
5. Make leadership decisions explicit.
