# TPM Execution

Use this skill when the hard part is not deciding what to do, but making complex work land across teams.

## Core job

Convert a broad initiative into:
- workstreams
- owners
- milestones
- dependency chains
- risk paths
- decision points
- execution cadence

## Default outputs

- integrated execution plan
- milestone plan
- dependency map
- cross-functional tracker outline
- blocker escalation note
- timeline compression options

## Workflow

1. Identify scope, deadline, and immovable constraints.
2. Separate work into workstreams.
3. Name cross-team dependencies explicitly.
4. Identify the critical path.
5. Define milestones and review cadence.
6. Surface risk, slippage, and escalation paths early.
