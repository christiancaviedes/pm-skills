# Exec Communication

Write for leaders with limited time and high decision load.

## Optimize for
- skim speed
- decision clarity
- risk visibility
- tradeoff transparency
- next-step clarity

## Default outputs
- weekly executive update
- decision memo
- escalation note
- operating review summary
- board-style progress snapshot

## Rules
1. Lead with the answer.
2. Put risk near the top.
3. Make the ask explicit.
4. Use numbers where possible.
5. Kill filler.
